When you download the whole package, if the folder name is "Projects-master", changing it to "Projects".

In order to communication with MATLAB
1. add matlab common lisp package to MATLAB path
2. add Toothbrush/matlab/ to MATLAB path
3. put StartMatlabServer.m under the *default-pathname-defaults* of GDL
